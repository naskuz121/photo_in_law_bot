Інструкція для запуску Telegram-бота на Railway:

1. Завантаж цей архів у новий проєкт на Railway (через "Deploy from GitHub" або "Upload Project").
2. У розділі Settings → Variables додай змінну середовища:
   BOT_TOKEN = твій токен Telegram-бота
3. У розділі Deployments → Start Command введи:
   python bot.py
4. Натисни "Deploy" — бот запуститься.

Гайд повинен бути завантажений окремо і посилання на нього має бути в коді.
